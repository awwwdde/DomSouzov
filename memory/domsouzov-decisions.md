Проект «Дом Союзов» (C:\Users\vlad\Documents\542\DomSouzov) — сайт культурного учреждения, замена старого https://domsojuzov.ru/. Прод-домен: **https://union.awwwdde.art/**. Сдача **1 июля 2026**. Стек: React 18 + TS + Vite + Tailwind v4 (SPA) + FastAPI + SQLite (dev) → **Postgres (prod)**, своя CMS-админка.

Порядок работ (уточнено 2026-06-05): 1) фиксы фронтенда+бэкенда и миграция на Postgres, 2) SEO, 3) доп.правки. Отдельная страница партнёров НЕ нужна — только секция на главной. Цель — полноценный релиз.

Решения пользователя (2026-06-05):
- **SEO**: пререндеринг публичных страниц при сборке (vite-плагин, напр. vite-react-ssg) + react-helmet-async для динамических meta-тегов. Новости/афиша должны находиться в поиске Яндекс/Google и корректно шариться. Нужны robots.txt, sitemap.xml, JSON-LD (Event/Article/Organization), og/twitter.
- **Акцентный цвет**: глубокий изумруд **#1F5F4E** (премиальность). Заменяет нынешний `--color-accent: #0a0a0a` в frontend/src/index.css.
- **PDF тех.райдера / залы**: загрузка PDF через админку (новое поле у Hall или отдельная настройка). Кнопки «просмотреть технический райдер» и «залы» на странице Организаторам открывают PDF в новой вкладке.
- **Объём**: сначала согласовать план, потом код.

Переработать страницы: Организаторам (минимум текста, без списков, 2 кнопки → PDF), Зрителям (правила нахождения, красивое оформление).
Баги: переход между страницами не прогружает контент (причина — Lenis + framer whileInView рассинхрон, секции остаются opacity:0); меню не адаптивно; битые ссылки в футере; мелкие лого партнёров.
